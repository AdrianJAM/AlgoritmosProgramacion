INTRUCCIONES DE USO
-----------------------------------

1. Para compiliar el archivo JAVA se debe:
1.1. Ingresar a la carpeta donde se encuentra el archivo .java con el comando cd
1.2. Escribir en la consola de comandos:
       javac pjt009.java
1.3. Ejecutar el archivo bycode .class generado en el momento de la compilación
       java Main
1.4. Interactuar con el programa según las preguntas que haga al usuario

NOTA:
- Antes de hace un Commit y un Push (subir al repositorio github) se debe eliminar todos los archivos bycode .class
